## Requirements

- [Rust](https://rustup.rs/)
- [Cap'n Proto](https://capnproto.org/)

Game simulation/back-end of Cao-Lo.

## Enabling state logging

- Compile with feature `log_tables`
- Set the log level of `caolo_sim::storage::views` to `Trace`
